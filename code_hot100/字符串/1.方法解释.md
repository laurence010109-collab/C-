1. t.find(s) 成员函数说明
/** * @brief 在容器中查找特定元素或子串
 * * @param s: 要查找的目标。
 * (1) 在 std::string 中: 可以是字符、字符串常量或 std::string 对象
 * (2) 在 std::set/std::map 中: 是要查找的 Key（键值）
 * @return: 
 * (1) 在 std::string 中: 返回目标首次出现的位置索引（size_t）。若未找到，返回 string::npos
 * (2) 在 std::set/std::map 中: 返回指向该元素的迭代器。若未找到，返回 t.end()
 */


2. t.erase(...) 成员函数说明
 /** * @brief 从容器中移除一个或多个元素
 * * @param 参数形式取决于调用方式:
 * (1) t.erase(iterator): 删除迭代器指向的单个元素
 * (2) t.erase(first, last): 删除 [first, last) 区间内的所有元素
 * (3) t.erase(key) [仅限关联容器]: 删除所有键值为 key 的元素
 * (4) t.erase(pos, len) [仅限 std::string]: 从索引 pos 开始删除 len 个字符
 * @return: 
 * (1) 序列容器 (string/vector): 返回指向被删元素下一个位置的迭代器
 * (2) 关联容器 (set/map): 返回被删除的元素个数（通常为 0 或 1）
 */
