#### 3.5.2 stack 常用接口
功能描述：栈容器常用的对外接口

构造函数：
* `stack<T> stk;`                                 //stack采用模板类实现， stack对象的默认构造形式
* `stack(const stack &stk);`            //拷贝构造函数


赋值操作：
* `stack& operator=(const stack &stk);`           //重载等号操作符


数据存取：
* `push(elem);`      //向栈顶添加元素
* `pop();`                //从栈顶移除第一个元素
* `top(); `                //返回栈顶元素


大小操作：
* `empty();`            //判断堆栈是否为空
* `size(); `              //返回栈的大小

总结：
* 入栈   --- push
* 出栈   --- pop
* 返回栈顶   --- top
* 判断栈是否为空   --- empty
* 返回栈大小   --- size

总结：
- 入队   --- push
- 出队   --- pop
- 返回队头元素   --- front
- 返回队尾元素   --- back
- 判断队是否为空   --- empty
- 返回队列大小   --- size

stoX 家族：字符串脱掉外壳变数字
stoi(s)：转成普通整数 (int)。
stoll(s)：转成超大整数 (long long)。如果你刷算法题遇到极大的数字，stoi 会爆表，必须用它！
stod(s)：转成双精度浮点数 (double)。处理带小数点的文本。
stof(s)：转成单精度浮点数 (float)。




数字穿上外衣变字符串 (to_string)
to_string(数字)：不管是 int、double 还是 long long，统统转成 std::string。




vector<string> words = {"hello", "world", "cpp"};
// const 保证绝对安全，& 保证绝对零拷贝！
for (const string& w : words) { 
    cout << w << endl;
}
加了 &：避免了拷贝字符串这种极其昂贵的内存搬运动作（零拷贝）。
加了 const：给这根绳子上了把锁，你在循环里只能看，绝对不能改。
万一你手抖写了 w = "bug"，编译器会当场把你拦住！

只需要读取：for (const auto& item : container)
需要原地修改：for (auto& item : container)
只拷贝小体积数据（如 int, char）：for (auto item : container)
